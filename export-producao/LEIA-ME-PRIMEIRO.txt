PASSO 1: Atualizar Footer
─────────────────────────
1. Acesse cPanel → File Manager
2. Vá até: public_html/cpanel/wp-content/themes/frederica-passos/
3. Faça backup do footer.php atual
4. Faça upload do novo footer.php desta pasta

PASSO 2: Importar Elementor (Escolha UMA opção)
────────────────────────────────────────────────

OPÇÃO A - Via WordPress Admin (Mais Fácil):
1. Acesse: https://fredericapassos.pt/cpanel/wp-admin
2. Vá em: Páginas → Home → Editar com Elementor
3. No Elementor: ☰ → Importar Template
4. Selecione: elementor-data.json
5. Clique em Atualizar

⚠️ OPÇÃO B - Via phpMyAdmin (NÃO RECOMENDADO - Pode desconfigurar o site!)
   Use apenas se a Opção A não funcionar e você souber o que está fazendo.
   Veja o arquivo SOLUCAO-SEGURA.md para instruções detalhadas.

PASSO 3: Limpar Cache
─────────────────────
1. WordPress Admin → Elementor → Tools → Regenerate CSS
2. Limpe cache do navegador (Ctrl+F5)
